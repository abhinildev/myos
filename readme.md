### Operating system created in rust programming language so far ###

This is a final year project being done by Abhinil, Reetam, Bishal and Nihar. So far project the project has the following features:
    1. Bootloading 
    2. Interrupt management 
    3. Memory management
    4. VGA text buffer
    5. Memory Management
    6. File System done
    7. Shell (ongoing)
    8. Context switching along with task management (planned)


## Setup
Run these commands in order
- `cargo install bootimage`
- `cargo build` (to check for build errors)
- `cargo run` (to run in qemu - qemu must be installed)
